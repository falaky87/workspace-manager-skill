# Workspace Manager Skill

Workspace organization assistant for [ClawPad](https://github.com/mhmdez/clawpad) and AI agents.

## Installation

```bash
npx skills add falaky87/workspace-manager-skill
```

## What it does

Helps users set up and maintain well-organized workspaces tailored to their needs:

- **Onboarding flow** - Guides new users through workspace setup
- **Domain templates** - Pre-built structures for:
  - Engineering & DevOps
  - Research & Academia
  - Business & Consulting
  - Creative & Writing
  - Personal Knowledge (PARA method)
- **Document templates** - Plans, tracking docs, runbooks
- **Naming conventions** - Consistent organization patterns

## Also available on ClawHub

```bash
clawhub install workspace-manager
```
